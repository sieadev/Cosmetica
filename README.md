# Cosmetica

Cosmetica is a (hopefully soon to be) fully customizable cosmetics Plugin.




## Features

- Pets
- Effects: Particles, Cloaks & more
- Costumes (Helmet, Chestplate, Leggins, Boots)
- Hats
- Sound Effects
